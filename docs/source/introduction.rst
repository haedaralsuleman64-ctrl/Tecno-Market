Introduction
============

Overview
--------

TEcNO Market is a comprehensive e-commerce platform built with Django that provides:

- User authentication and authorization
- Product catalog management
- Shopping cart functionality
- Order processing system
- Payment integration
- Admin dashboard

Features
--------

- **User Management**: Registration, login, profile management
- **Product Catalog**: Categories, products with images and descriptions
- **Shopping Cart**: Add/remove items, quantity management
- **Order System**: Checkout process, order history
- **Admin Panel**: Complete backend management
- **Responsive Design**: Works on desktop and mobile devices

Technology Stack
----------------

- **Backend**: Django 5.2.5
- **Database**: SQLite (development), PostgreSQL (production)
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Authentication**: Django Allauth
